# PLAYHAUZ Neon Glitch (Static Demo)

## Run locally
- Open `index.html` directly in browser **OR**
- Use a simple local server (recommended):
  - VS Code: install "Live Server" extension and click "Go Live"
  - Python:
    - `python -m http.server 8000`
    - open http://localhost:8000

## Deploy (Netlify / Render static)
- Upload the folder as a static site.
- Build command: (none)
- Publish directory: the folder root (where index.html is)

## Replace images
Replace files inside `/assets` (keep the same names) or update `index.html`.
